You can use the link below to download a tar file which contains:

  • A JSON file containing all moderated annotations
  • A live snapshot of the static site content

Vaguely in my mind this feels like a safety net because the annotations are tied to the precise layout of the html of the site. So if you have a copy of the site exactly as is and a copy of all the annotation data at a given moment, they ought to definitely match each other. Of course, if the layouts were to change, then it would I think knock the annotation data in the DB out of whack, so really major changes to the textual layout should be rare and accompanied by some kind of recalibration of the annotation ranges. So I'm hoping that never needs to happen.
